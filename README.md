## test backend 
Изпользованы: nest.js postgreSQL TypeORM      


### Настройки по умолчанию
      type: 'postgres',  
      host: 'localhost',  
      port: 5432,  
      username: 'test_user',  
      password: 'password',  
      database: 'test_project',  
      synchronize: true, 


### сборка проекта 
nest build 

### запуск проекта 
nest start 

### документация в swagger достпна после запуска проекта
{url}/api
